# Notes Backend

## To set up

- `npm install`
- `npm install express`
- `npm install --save-dev nodemon`

## To run / Develop

- `npm run dev`
- Add the following scripts to your `package.json`

```js
"scripts": {
    "start": "node index.js",
    "dev": "nodemon index.js",
    "test": "echo \"Error: no test specified\" && exit 1"
  },
```
